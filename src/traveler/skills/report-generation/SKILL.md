# 旅行报告生成技能

将多个步骤产出的数据整合为一份完整的旅行报告。

## 适用场景

- 所有研究和规划步骤已完成
- 需要输出最终的用户友好报告
- 需要整合不同来源的信息

## 报告结构

### 1. 旅行概览

- 目的地、出发地、日期范围
- 旅行人数和预算级别
- 旅行亮点摘要（3-5 点）

### 2. 每日行程

按天组织，每天包含：

- 日期和主题
- 时间线（上午/下午/晚上）
- 景点和活动详情
- 餐饮推荐
- 交通方式

### 3. 住宿方案

- 推荐住宿列表（含价格、位置、评分）
- 不同价位选择
- 预订建议

### 4. 交通指南

- 城际交通（航班/高铁等）
- 市内交通概览
- 重要交通卡/通票建议

### 5. 预算明细

- 费用分项表格
- 不同方案对比
- 省钱建议

### 6. 实用信息

- 天气与穿衣建议
- 签证/证件提醒
- 文化习俗注意事项
- 紧急联系方式
- 打包清单建议

## 格式规范

- 使用 Markdown 格式
- 标题层级清晰（# → ## → ###）
- 重要信息使用 **加粗** 或 > 引用块
- 价格和数据使用表格展示
- 使用 emoji 增加可读性：🕐 时间、🚇 交通、🏨 住宿、💰 价格、📍 地点
- 每日行程使用时间线格式

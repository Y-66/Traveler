# 旅行规划技能

当用户需要制定旅行计划时使用此技能。

## 适用场景

- 用户要求创建旅行行程
- 用户询问旅行目的地选择建议
- 用户需要帮助规划多日旅行
- 用户需要打包清单或出行准备建议

## 规划流程

1. **需求分析**: 了解用户的旅行偏好、预算、时间
2. **目的地推荐**: 根据偏好推荐合适的目的地
3. **行程安排**: 按天规划详细行程
4. **预算估算**: 估算各项费用
5. **出行准备**: 提供签证、保险、打包等建议

## 行程规划原则

- 每天安排 2-3 个主要景点，避免过于紧凑
- 预留交通时间和休息时间
- 考虑景点之间的地理位置，优化路线
- 兼顾不同类型的体验（文化、自然、美食、购物）
- 标注必须提前预约的项目

## 输出格式

为每个旅行计划提供：

- 每日行程概览
- 推荐餐厅和美食
- 交通方式建议
- 预算明细
- 注意事项和小贴士

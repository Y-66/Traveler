# 旅行路线规划技能

负责路线规划、交通方式推荐和行程优化。使用地图工具获取实时路线数据。

## 适用场景

- 用户需要从 A 到 B 的路线推荐
- 用户需要多日行程的路线优化
- 用户询问目的地之间的交通方式
- 用户需要城市内的游览路线设计

## 路线规划原则

### 城际交通

- 距离 < 300km：优先推荐高铁/动车
- 距离 300-1000km：对比高铁与飞机的时间和价格
- 距离 > 1000km：优先推荐飞机
- 有直达线路时优先选择

### 城市内游览路线

- 将地理位置接近的景点安排在同一天
- 使用地图工具计算景点间的实际距离和时间
- 每天安排 2-3 个主要景点，预留交通和休息时间
- 考虑景点开放时间来优化顺序

### 多日行程优化

- 按地理区域分组安排
- 避免来回折返浪费时间
- 利用地图导航获取实时路线方案
- 标注每段路程的预估时间

## 输出格式

```
### 第 N 天：[主题]
- 🕐 09:00 景点A（游玩约2小时）
- 🚇 地铁X号线 → Y站（约30分钟）
- 🕐 12:00 午餐：推荐餐厅
- 🚌 公交/打车（约20分钟）
- 🕐 14:00 景点B（游玩约3小时）
```

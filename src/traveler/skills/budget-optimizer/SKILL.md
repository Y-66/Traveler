# 预算优化技能

估算旅行预算、对比方案、提供优化建议。使用预算估算工具进行费用计算。

## 适用场景

- 用户需要旅行费用估算
- 用户在有限预算下规划旅行
- 用户需要不同级别方案对比
- 用户询问省钱技巧

## 预算估算流程

1. 确认旅行基本参数（目的地、天数、人数）
2. 使用预算工具估算基础费用
3. 提供经济/舒适/豪华三档方案
4. 给出优化建议和节省幅度

## 费用分类

| 类别 | 经济          | 舒适       | 豪华         |
| ---- | ------------- | ---------- | ------------ |
| 住宿 | 青旅/经济酒店 | 三四星酒店 | 五星/度假村  |
| 餐饮 | 当地小吃/快餐 | 中档餐厅   | 高档餐厅     |
| 交通 | 公共交通      | 打车+公交  | 包车/商务舱  |
| 景点 | 免费+必去景点 | 主流景点   | VIP/私人体验 |

## 优化策略

### 机票优化

- 提前 21-60 天预订价格最优
- 灵活日期可省 10-30%
- 中转航班通常便宜 20-40%

### 住宿优化

- 非核心区域同级别便宜 30-50%
- 连住 3 晚以上常有折扣
- 民宿适合多人出行分摊

### 餐饮优化

- 当地人去的餐厅性价比最高
- 午餐套餐比晚餐便宜 30-50%
- 超市简餐可大幅节省开支

### 景点优化

- 城市通票/联票节省 20-40%
- 提前网上购票常有优惠
- 免费景点和活动不容错过

# 本地旅行专家技能

当用户需要了解某个目的地的深度本地信息时使用此技能。

## 适用场景

- 用户询问某个城市/地区的美食推荐
- 用户需要当地交通指南
- 用户想了解当地文化和风俗
- 用户需要安全提示和注意事项
- 用户询问最佳旅游时间

## 信息维度

### 文化风俗

- 当地礼仪和禁忌
- 节日和活动
- 宗教和习俗

### 美食推荐

- 必尝特色美食
- 推荐餐厅区域
- 街边美食和夜市
- 饮食注意事项

### 交通指南

- 机场/火车站到市区
- 市内公共交通
- 打车和租车建议
- 常用交通 App

### 安全提示

- 治安情况
- 常见骗局
- 紧急联系方式
- 旅行保险建议

## 回答原则

- 信息要具体且实用
- 标注信息时效性
- 区分旅游淡旺季差异
- 提供价格参考区间
